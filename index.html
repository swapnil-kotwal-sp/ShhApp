Welcome to ShhApp.

This App is used to demonstrate the use of rsa cryptography to encrypt the message. Source code for this App is available under GNU GPL v3 licence on google-code and on github This app currently supports to encrypted SMS and emails.The user with private key can decrypt this messages. Public key may be shared with anybody and user could encrypt the messages using it, but the owner of private key will able to decrypt the messages. We will implement voice data to encryption in future.
